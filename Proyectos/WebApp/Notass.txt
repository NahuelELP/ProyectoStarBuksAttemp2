[FromBody] Usuario usuario

👉 es como decir:

“creame un objeto Usuario con los datos que vienen en el JSON”
Para que entiendas el panorama:

[FromBody] → body (JSON)
[FromRoute] → URL (/usuarios/1)
[FromQuery] → query string (?nombre=juan)
[FromHeader] → headers
